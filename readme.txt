Обучил нейронную сеть yolo 11 версии для нахождения 5 различных цветов на картинках

Чтобы обучить ее:
в папке prepare_dataset программа которая перераспределяет dataset/flowers в структуре 20 на 80 по нуждам yolo
после запускаем код (на всякий проверяем путь до папки flowers_prepared)

Чтобы просто протестировать:

в папке main code файл с обучением нейронки и файл find_object который определяет есть ли цветы на картинке
чтобы протестировать на своей картинке скачайте любое изображение и засуньте в папку test_dir и измените путь до файла
